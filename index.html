Nutsacker-extension

A browser extension for Chrome, Safari and Opera Next that adds a jaunty pair of balls to every image on the current page at the click of a button

Welcome to Nutsacker.

Download and Install

For Safari, you have to download the .safariextz file and double-click it to install. Chrome and Opera Next can install it from the Chrome Web Store.

Usage

After install, press the button to Nutsack the images!
